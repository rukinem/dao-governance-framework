# 🏛️ dao-governance-framework

Modular DAO governance framework with proposal, voting and execution

## Features

- Proposal creation
- Voting mechanisms
- Execution timelock
- Modular design
- Access control

## Tech Stack

- **Smart Contracts:** Solidity ^0.8.20
- **Framework:** Hardhat
- **Frontend:** Next.js + TypeScript
- **Web3:** ethers.js / wagmi

## Getting Started

```bash
npm install
npx hardhat compile
npx hardhat test
npx hardhat run scripts/deploy.js --network localhost
```
